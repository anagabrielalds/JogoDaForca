Regras do Jogo

O objetivo deste jogo é descobrir qual é a palavra adivinhando as letras que ela possui. O jogador deve usar o teclado para sugerir uma letra que suspeitem fazer parte da palavra. Caso a palavra contenha esta letra, será mostrado em que posição(ões) ela está. Entretanto, caso esta letra não exista na palavra, será desenhada uma parte do corpo do boneco do jogador e aparecera no campo de palavras erradas. Se o jogador errar as letras 8 vezes, ele perderá o jogo e caso encontre a palavra ele vencerá
